Hexo-Ubuntu16.04下安装Node.js

安装Node.js 在搭建Hexo博客之前，要要先安装Git和Node.js。Ubuntu16.04 虽然自带的有， 版本太低，且没有npm工具，直接 sudo apt install nodejs 会提示需要安装依赖，比较麻烦。 简单的办法是指定版本更新，可去Node的官网查看版本(不建议选择太旧的版本)，然后将下面第一行shell的数字10替换成相应版本号即可： 1$ curl -sL
