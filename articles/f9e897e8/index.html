博客网站搭建过程

Lukelqp

2019-07-20

Hexo

使用Hexo 生成静态网页的功能可以搭建一个免费的博客网站，以下步骤在 Windows 10 和 sublimetext3 环境下完成，Linux下的安装请参考Linux上安装Node.js，后续操作类似。

安装Hexo平台

  • 在安装hexo之前，需要先安装Git和Node.js依赖。

安装Git

  • Git 是分布式版本控制系统，可以把本地的网页和文章等内容提交到 GitHub 上，实现同步。 Mac 和 Linux 可能会自带 Git，Windows 则需要自行安装。

  • 点击 下载地址 下载安装包，安装选择默认就行，安装后，打开命令行窗口，输入：

安装Node.js环境

  • Node.js是基于 Chrome V8 引擎的 JavaScript 运行环境，可以在非浏览器环境下，解释运行 JS 代码。 Node 带有 npm 包管理工具，可以用来安装 Hexo。

  • 前往下载，依然是选择默认安装，然后在命令行测试安装：

  • npm是国外源，网速不稳定，建议换国内源，文章参考关于包管理工具npm下载速度慢的问题。

部署到GitHub

创建 GitHub 仓库

  • 搭建个人博客需要一个域名，可以申请购买一个域名来使用。也可以通过 Github Pages 获得一个免费使用的二级域名。

  • 首先在 Github 上新建一个仓库，新仓库的名字一定要是 UserName.github.io 的形式，UserName 就是用户名，建议取个好的用户名。这样最后形成的博客的网址就是 https://UserName.github.io

    提示：固定新仓库的名字格式并非必须，只是这样操作生成的博客域名比较短小简洁，另起他名生成博客域名会很冗长。

  • 点击 Create Repository 新建仓库之后，选择 settings，找到 GitHub Pages 下面的 choose theme ，选择一个主题后点 select。

  • 再去查看 GitHub Pages 下面的 choose theme ，发现多了一个 Custom domain ，这个时候就能使用 https://UserName.github.io 这个域名了。(图片来自网络)

基本设置

更换主题

  • Hexo 有很多主题， NexT 是一个非常流行的博客主题，喜欢简单风格的可以试一下。

  • 在博客根目录下打开 /myblog/_config.yml ，修改主题 theme: next ，部署即可看到效果。

  • NexT 还支持很多自定义设置，详情参考博客页面优化(一)、博客页面优化(二)、更多主题。

设置菜单

  • 找到主题配置文件/myblog/themes/next/_config.yml，查找menu，去掉#注释即可显示对应的菜单项。
