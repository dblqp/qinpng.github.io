  • 推荐在谷歌浏览器的扩展商店里添加新浪微博图床，拖动即可生成图片链接。

文章唯一地址

  • 在默认设置下，更改文章题目或者变更文章发布时间，文章链接都会改变，不利于搜索引擎收录和分享。而且如果链接的层级太深(即太长)，对SEO不友好。所以简短的、唯一永久链接是更好的选择。

  • 安装插件cnpm install hexo-abbrlink --save
